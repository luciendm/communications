Dispatcher Service Recovery Guide

Quick reference for handling service issues and escalations

📞 First Level Resolution (FLR)

  1. Apologize & acknowledge the member’s experience.
  2. Review documentation and provide ETA updates.
  3. Explain the actions being taken.
  4. Thank the member and close professionally.

⚠️ Active Calls — Formal Complaint

  1. Apologize and follow FLR.
  2. Offer to Allow the call.
  3. If immediate escalation requested → warm transfer to SRT or your Local Supervisor Line.

✅ Allowing Calls

  1. Open RADAR → Case Actions.
  2. Select Allow → Next.
  3. Check Allow Flag.
  4. Select reason.
  5. Add documentation.

Plus/Premier: Supervisor override required.

📝 Inactive Calls

  1. Create CMS case (Dispute of Call Count).
  2. Select Call Member Back.
  3. Confirm contact information.
  4. Document reason in comments.

🗂️ Completed / Cancelled Calls

  1. Apologize for the inconvenience.
  2. Offer to file complaint with station management.
  3. Advise member the responsible party will follow up.
  4. Process complaint in CMS per ERS protocols.

🧭 Decision Helper

Choose options and select “Get Guidance”.
